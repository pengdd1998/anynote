#### Flutter Test Fixes (27 failing → 0 failing)
Fixed Drift StreamQueryStore timer leaks across 16 test files:

**Root Cause**: Using `addTearDown()` for TestAppHandle disposal caused the test framework to check for pending timers before tearDown runs, resulting in "A Timer is still pending" errors.

**Solution**: Manual disposal pattern at end of test body:
```dart
await handle.dispose();
await tester.pump(const Duration(milliseconds: 100));
```
